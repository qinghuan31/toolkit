# 分容分析插件 版本更新日志

本文件为分容分析插件（capacity_analysis）独立更新日志。

---

## [1.8.0] - 2026-06-17

### 新增功能
- **分容数据统计分析插件**（`plugins/capacity_analysis/`）：
  - 支持精捷能分容柜导出格式（Excel/CSV）
  - 自动识别"工作状态"行 + "数据内容"表头
  - 工步关键字匹配（恒流放电 / 恒流充电），不依赖固定工步号
  - 工艺模式识别：充电 → 放电 → 充电（普通分容），剔除循环测试、剔除先放后充
  - 取最后一次"恒流放电"工步的"单步容量(mAh)"作为电池容量
  - 异常样本分类：无放电 / 无充电 / 顺序错误 / 循环测试 / 空值
- **JMP 风格分布图渲染**（matplotlib）
  - 横向直方图（绿色填充）
  - 垂直箱体图（含中位数线、Q1/Q3 箱、上下须、离群点）
  - 分位数表（11 行百分位 + 3 行标签值）
  - 汇总统计表（均值 / 标准差 / 95% 置信区间 / 样本数）
  - 单图 PNG，1440×800，150 DPI
- **统计指标计算**（与 JMP 一致）
  - 均值 / 标准差 / 中位数 / 最小最大 / Q1 / Q3 / 2.5% / 97.5%
  - 95% 置信区间（基于 t 分布，自由度 n-1，无需 scipy 依赖）
  - 均值标准误差
- **UI 集成**
  - 一级侧边栏入口（自动注册）
  - 工具栏：选择文件 / 选择文件夹 / 开始分析
  - 实时日志区
  - 统计概览区（10 项指标）
  - 异常样本查看按钮（带分类明细）
  - 出图按钮（让用户每次选目录）
